    What is HLNoobs.net?

    The HLNoobs Project is a passion project ran by Rizapiza and f0rest_crypt1d that gets contributed to by group members. HLNoobs.net is our Website, Game Servers, Community, and where the HLNoobs Project publicly hosts it's info. We try to find and make guides and resources for Source and GoldSrc game server hosting, modding, valve software,

    Want to help the HLNoobs Project?

    There are currently 2 options for helping us.

    Option 1: Join our Discord Server and look the the #how-to-help-hlnp channel to see what kind of help we are actively looking for, maybe theres something we can point you towards.

    Option 2: Donate to the HLNoobs Project so we can continue to host servers and keep adding resources to HLNoob.net's info section.

    We are working on setting up a donations page
